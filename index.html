🏎 타이어핑 최저가 봇 v1.0

경쟁가 자동조회 · 가격변경 · 상품 자동등록 크롬 확장

⚠️ 타이어핑 계정 보유자만 사용 가능합니다.
크롬에서 www.tireping.com 에 먼저 로그인 후 사용하세요.

📦 최초 설치 방법

⬇ ZIP 다운로드
  1. 위 ZIP 파일 다운로드 후 압축 해제 → tireping_ext 폴더 생성
  2. 크롬 주소창에 chrome://extensions 입력 후 Enter
  3. 우측 상단 개발자 모드 켜기
  4. "압축해제된 확장 프로그램을 로드합니다" 클릭
  5. tireping_ext 폴더 선택
  6. 타이어핑 로그인 후 확장 아이콘 클릭 → 사이드패널 열림 ✅

🔄 업데이트 방법

  1. 크롬 주소창에 chrome://extensions 입력
  2. 우측 상단 개발자 모드 ON 확인
  3. 좌측 상단 업데이트 버튼 클릭 → 자동 완료 ✅

※ 업데이트 공지 후 위 방법대로 진행해주세요.
